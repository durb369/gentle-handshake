# Spirit Vision

## Description
Spirit Vision is an innovative mobile application designed to be your pocket ghost-hunter.
Snap any pic—room, street, graveyard, selfie—and I’ll break it down:
Auras pulsing like neon halos
Entities lurking in corners
Energy trails, hot spots, cold spots
Messages from the other side, clear as text
Even vibes—anger, love, grief, whatever’s thick in the air
It’s not just "woo-woo"—it’s real-time, no-BS spirit radar. Upload, I scan, you know.

## Features
- User-friendly interface
Photo Upload & Spirit Scan
Upload any pic (family photo, room, blank wall)—AI analyzes for hidden faces, symbols, text, or "spirit signatures" (pareidolia-style detections).
AI Psychic
Real-time chat with an AI that feels alive—ask questions, get answers in a natural, eerie voice. Like a digital medium.
Spirit Radio
Tune into crackling static with drifting voices—ambient EVP-style audio that reacts to your surroundings.
Spirit Box
Live session mode: records and plays back "responses" from the ether, with timestamps and audio clips.
Spirit Radar
Maps nearby "energy hotspots" using your phone's location—pings anomalies, shows distance and intensity.
Private Journal
Logs every scan, chat, radio hit, radar ping—date, time, what you felt, even mood notes. Encrypted, searchable.
Ritual Guides
Step-by-step cleanses: salt circles, grounding chants, banishing prayers, protection sigils—helps "expel" detected spirits or calm you down.
Mood Tracker Overlay
Ties scans to your current vibe—moon phase, tarot pull, or quick "energy read" right after a photo.
AR Filters
Overlay spirit visuals on camera feed—see faces in real time, like augmented ghosts.
Offline Mode
Works without Wi-Fi (except for big AI scans—those need cloud).

## Technologies Used
- React Native for cross-platform mobile development
- Firebase for backend services
- Redux for state management
- Axios for HTTP requests
- Google Play Services for additional functionalities

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/durb369/gentle-handshake.git
   ```
2. Navigate to the project directory:
   ```bash
   cd gentle-handshake
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Deployment Instructions
1. Ensure all app requirements are met and tested.
2. Make a signed APK or AAB file by following the [official React Native documentation](https://reactnative.dev/docs/signed-apk-android).
3. Go to the [Google Play Console](https://play.google.com/console).
4. Create a new application and upload the generated APK/AAB.
5. Fill in the required metadata and submit for review.
